# 📊 System Monitor (s) — Легкий монитор сервера

**System Monitor** — ультракомпактный Bash-скрипт для мониторинга ресурсов в реальном времени. Вся инфографика сервера (CPU, RAM, Сеть, Docker) по одной короткой команде `s`. 🚀

---

### 📥 Быстрая установка

```bash
curl -sSL https://raw.githubusercontent.com/zazdravie/sys-monitor/main/install.sh | bash
```

### 🛠 Управление и флаги

| Команда | Описание |
| --- | --- |
| `s` | ⚡ **Мгновенная сводка**: Текущее состояние ресурсов и сети. |
| `s --autostart` | 🔄 **Автозапуск**: Включает монитор при каждом входе по SSH. |
| `s --autostart-remove` | 🛑 **Отключить автозапуск**: Удаляет скрипт из `.bashrc`. |
| `s --uninstall` | 🗑 **Удаление**: Полная очистка системы от команды `s`. |

---

### ✨ Ключевые фишки

* 🌍 **Network & Geo**: Внешний IP, город, страна и Real-time трафик (↓/↑ Kb/s).
* 📈 **Ресурсы**: Точная нагрузка **CPU** (через `/proc/stat`), **RAM/Swap** и остаток места на **SSD**.
* 🐳 **Docker**: Быстрый чек количества запущенных контейнеров.
* 🛡️ **Безопасность**: Статус Firewall и счетчик **SSH Fail** (защита от брутфорса).
* ⏱ **Uptime**: Время аптайма сервера в один клик.

### 📦 Требования

Минималистичный стек: `bash`, `curl`, `awk`, `sed`, `grep`, `free`, `df`. Работает «из коробки» на большинстве дистрибутивов.
