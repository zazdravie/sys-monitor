
# 📊 System Monitor (s) — Легкий монитор сервера на Bash


**System Monitor** — это компактный и быстрый инструмент для мониторинга ресурсов сервера в реальном времени. Он идеально подходит для тех, кто хочет видеть состояние системы (CPU, RAM, Сеть, Docker) сразу при входе по SSH или по нажатию одной клавиши.

---

## 🚀 Быстрая установка

Чтобы установить монитор и настроить команду `s`, просто введите в терминале:

```bash
curl -sSL https://raw.githubusercontent.com/zazdravie/sys-monitor/main/install.sh | bash

```

---

## 🛠 Управление и команды

После установки вам доступна команда `s`. Вы можете использовать следующие флаги для управления монитором:

| Команда | Описание |
| --- | --- |
| `s` | **Мгновенная сводка**: Показать текущее состояние ресурсов и сети. |
| `s --autostart` | **Автозапуск**: Монитор будет автоматически запускаться при каждом входе в систему (через `.bashrc`). |
| `s --autostart-remove` | **Отключить автозапуск**: Удаляет скрипт из автозагрузки, но оставляет возможность запускать его вручную. |
| `s --uninstall` | **Полное удаление**: Удаляет команду `s` и все связанные файлы из системы. |

---

## ✨ Основные возможности

Скрипт собирает данные из системных ресурсов и внешних API для предоставления полной картины:

* 🚀 **Мгновенный отклик**: Команда из одной буквы `s` экономит ваше время.
* 🌍 **Геопозиция и Сеть**:
* Определение страны, города и внешнего IP-адреса.
* Мониторинг скорости сети в реальном времени (↓ Прием / ↑ Отдача в Kb/s).


* 📈 **Мониторинг ресурсов**:
* **CPU**: Точный расчет нагрузки (через дельту `/proc/stat`).
* **RAM/Swap**: Использование оперативной памяти и подкачки.
* **SSD**: Свободное и занятое место на основном диске.


* 🐳 **Docker**: Подсчет количества запущенных контейнеров.
* 🛡️ **Безопасность**:
* **SSH Fail**: Подсчет неудачных попыток входа за текущие сутки (защита от брутфорса).
* **Firewall**: Проверка статуса защиты (UFW или Firewalld).


* ⏱ **Uptime**: Отображение времени непрерывной работы сервера.
