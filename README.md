# 📊 System Monitor (s)

Легкий и быстрый системный монитор для Linux/WSL на Bash. Показывает всё самое важное о сервере в одном компактном блоке при входе в систему или по команде.

---

## ✨ Основные возможности
* 🚀 **Мгновенный запуск**: Команда из одной буквы `s`.
* 🌍 **Геопозиция**: Показывает страну, город и внешний IP.
* 📈 **Ресурсы**: Загрузка CPU (с расчетом дельты), RAM, Swap и SSD.
* 🐳 **Docker**: Количество активных контейнеров.
* 🛡️ **Безопасность**: Статус Firewall (UFW/Firewalld) и количество неудачных попыток входа по SSH за текущие сутки.
* 📊 **Сеть**: Текущая скорость приема и отдачи (Kb/s).
